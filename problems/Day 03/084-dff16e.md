## What the question is asking

Build a 16-bit register with synchronous reset and independent byte-enable controls for its upper and lower bytes.

## Saved Verilog solution

```verilog
module top_module (
    input clk,
    input resetn,
    input [1:0] byteena,
    input [15:0] d,
    output reg [15:0] q
);
    always @(posedge clk)begin
        if(!resetn)begin
            q<=16'd0;
        end
        else begin
            if(byteena[1])
                q[15:8]<=d[15:8];
            if(byteena[0])
                q[7:0]<=d[7:0];

        end

    end
endmodule
```
