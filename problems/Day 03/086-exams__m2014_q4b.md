## What the question is asking

Model the positive-edge-triggered D flip-flop shown in the exam circuit.

## Saved Verilog solution

```verilog
module top_module (
    input clk,
    input d,
    input ar,   // asynchronous reset
    output q
);
    always @(posedge clk or posedge ar)begin
        if(ar)begin
            q<=0;
        end else begin
            q<=d;
        end
    end
endmodule
```
