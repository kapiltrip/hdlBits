## What the question is asking

Derive a minimal combinational expression from the supplied 4-variable Karnaugh map.

## Saved Verilog solution

```verilog
module top_module(
    input a,
    input b,
    input c,
    input d,
    output out
);
    assign out = c & d & (a | b ) | ~b & ~c | ~a & ~ d;
endmodule
```
