## What the question is asking

Build a 3-input lookup table from a shift register and multiplexer so its truth table can be loaded serially.

## Saved Verilog solution

```verilog
module top_module (
    input clk,
    input enable,
    input S,
    input A, B, C,
    output Z
);
//8 bit shift register
    reg [7:0] Q;
    always @(posedge clk)begin
        //S is the input to the shift, reg
        if(enable)begin
            Q[0]<=S;
            Q[7:1]=Q[6:0];
        end
    end
    always @(*)begin
        case({A,B,C})
            3'b000:Z=Q[0];
            3'b001:Z=Q[1];
            3'b010:Z=Q[2];
            3'b011:Z=Q[3];
            3'b100:Z=Q[4];
            3'b101:Z=Q[5];
            3'b110:Z=Q[6];
            3'b111:Z=Q[7];
        endcase
    end
endmodule
```
