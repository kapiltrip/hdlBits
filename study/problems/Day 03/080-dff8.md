## What the question is asking

Create eight D flip-flops in parallel so an entire 8-bit vector is registered on the rising clock edge.

## Saved Verilog solution

```verilog
module top_module (
    input clk,
    input [7:0] d,
    output reg [7:0] q
);
    always @(posedge clk)begin
        q<=d;
    end
endmodule
```
