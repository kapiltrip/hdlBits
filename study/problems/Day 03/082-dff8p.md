## What the question is asking

Create an 8-bit register whose synchronous reset loads the specified non-zero reset value.

## Saved Verilog solution

```verilog
module top_module (
    input clk,
    input reset,
    input [7:0] d,
    output reg [7:0] q
);
    always @(negedge clk )begin
        if(reset)
            q<=8'h34;
        else
            q<=d;
    end
endmodule
```
